Home Automation System Using ESP8266

  • This project is Home Automation System that utilizes the ESP8266 microcontroller and other electronics components to control various household appliances and devices remotely. The system can be accesed via web browser or mobile application, allow user to turn on/off light, fans, air conditioner and other appliances with a simple click. The ESP8266 is used as the main controller, connected to various sensor and relays to control the appliances. The system is connected to the local Wi-Fi network and can be accessed via web browser or mobile application. Provides a great opportunity to learn about microcontrollers, programming, and electronics. Helps to reduced energy consumption and costs by allowing users to turn off appliances when not in use.
